# Icon Assets

SVG source assets for the Guition JC4827W543 LVGL UI.

- Canvas-sized weather icons use `56x48`, matching the current weather icon area.
- `status-moon.svg` uses `42x42`, matching the current sun/moon status object.
- `status-offline.svg` uses `88x88`, matching the current Klipper offline ring area.
- These files are source assets only. The sketch code has not been changed.

For LVGL on the ESP32-S3, keep these SVGs as editable originals and convert selected icons to LVGL C image arrays later if needed.
